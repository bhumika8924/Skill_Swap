## 🚀 Overview
**SkillSync** is a community-driven platform designed for students and professionals to trade their skills without exchanging money. Instead of paying for expensive courses, you can find someone who knows what you want to learn, and teach them a skill you excel at in return.

**🌍 Live Demo:** [https://skillswap-4tg3.vercel.app/](https://skillswap-4tg3.vercel.app/)

## ✨ Features
- **Skill Discovery**: Browse and search through comprehensive student profiles categorized by their offered and desired skills.
- **Matchmaking**: Seamlessly view what others can teach and what they want to learn.
- **Profile Dashboard**: Personalize your profile, manage your bio, and highlight your core competencies.
- **Messaging System**: Interact with other members safely via the built-in communication panel.
- **Responsive UI**: A sleek, dark-themed, glassmorphism-inspired UI designed to provide a premium feel on any device.

## 🛠️ Tech Stack
- **Frontend Framework**: React.js 19
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Styling**: Modern Vanilla CSS and Inline System

## 🗂️ Project Structure

## 🔧 Installation & Setup
To run this project locally, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

1. **Clone the repository** (if not already local):
   ```bash
   git clone https://github.com/Yashika-28/skillswap.git
   cd skillswap
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm start
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

4. **Build for production**:
   ```bash
   npm run build
   ```
